# Examples how to use

### Simple display of the ColorPicker component
```javascript
import { ColorPicker } from 'mui-color';

export const Container = () => (
  <div>
    <ColorPicker defaultValue="transparent"/>
  </div>
);
```

```javascript
import { ColorPicker } from 'mui-color';

export const Container = () => (
  <div>
    <ColorPicker defaultValue="transparent" variant="outlined" fullWidth  />
  </div>
);
```
